### Running the Script
1. **Basic Scan**:
   ```bash
   python talons_blue.py --url https://example.com --domain example.com
   ```

2. **JSON Output**:
   ```bash
   python talons_blue.py --url https://example.com --domain example.com --json
   ```

3. **Custom Delay**:
   ```bash
   python talons_blue.py --url https://example.com --domain example.com --delay 3
   ```
